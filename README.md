# [飲み物の瓶追加]MoreBottles

[**ダウンロード**](https://github.com/eyeq/mod-1.11.2-MoreBottles/releases/download/1.0/1.11.2-MoreBottles-1.0.jar)

牛乳入り瓶：付与されている全てのポーション効果を打ち消します。  
溶岩入り瓶：飲むと発火します。  
延長のポーション：付与されている全てのポーション効果を10秒間延長します。  
変換のポーション：付与されている全てのポーション効果の時間と効能を交換します。  
伝播のポーション：付与されている全てのポーション効果の弱体版を周囲に与えます。
